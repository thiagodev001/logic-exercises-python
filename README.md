# Desafios de Lógica de Programação com Python

Repositório destinado ao estudo de algoritmos e estruturas de dados fundamentais utilizando a linguagem Python.

## 💻 Conteúdo do Repositório
Este repositório está organizado por nível de complexidade e conceitos aplicados:
- **Básico:** Operadores aritméticos, entrada/saída de dados e manipulação de strings.
- **Estruturas Condicionais:** Verificadores de paridade, validadores de dados e sistemas de decisão.
- **Estruturas de Repetição:** Algoritmos de busca em listas, geradores de sequências e automação de cálculos.
- **Funções e Módulos:** Modularização de código para reaproveitamento e organização.

## 🎯 Objetivo
Desenvolver uma base sólida em algoritmos para aplicação em desenvolvimento **Backend** e scripts de automação de segurança.

## ⌨️ Como executar
1. Certifique-se de ter o Python 3.x instalado.
2. Clone o repositório: `git clone https://github.com/thiago-dev-x/logic-exercises-python.git`
3. Execute os arquivos: `python nome_do_arquivo.py`
